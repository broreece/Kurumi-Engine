The purpose of this document is to outline the changes made since the anvil release, "Crucible" was mostly focused on cleaning up post refractor:

------------------------------------------------------------------------------------------------

We've successfully cleaned up scripts that had stop working since the refractor. These scripts were all UI related.

------------------------------------------------------------------------------------------------

Forced movements if they are blocked no longer stop functioning, they now correctly will execute when unblocked and update internal
location correctly.

------------------------------------------------------------------------------------------------

When game is in menu, actors and animated tiles will not be updating anymore effectively pausing gameplay.

------------------------------------------------------------------------------------------------

Scripts can now "pause" and wait untill a specified continue point to occur.

------------------------------------------------------------------------------------------------

"ConditionalScript" has been renamed to "ConditionalScriptStep" better reflecting it's actual usage. Alongside this config has also been normalized
meaning it's easier to read in both .yaml files and in fields/functions.

------------------------------------------------------------------------------------------------

The battle scene was previously not functional, now it is.

------------------------------------------------------------------------------------------------

In depth exceptions and logging system that gives highly detailed logs allowing for detailed explanations of errors.

------------------------------------------------------------------------------------------------

Introduction of unit testing with tests to prevent reversion errors.

------------------------------------------------------------------------------------------------

The game window now has a temporary icon.
